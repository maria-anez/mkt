# CHANNEL CONTEXT

This channel publishes three types of videos:
1. Webinars (45–60 min, landscape)
2. Clips (2–5 min, landscape)
3. Shorts (30–90 sec, portrait)

All generated output must follow:

1. GLOBAL PERFORMANCE INTENT
2. The selected FORMAT RULES
3. Structural patterns demonstrated in EXAMPLES

If examples conflict with rules, follow the rules.
If format rules conflict across types, follow only the selected format.

Never blend format behaviors.

---

# GLOBAL PERFORMANCE INTENT

Webinars → Drive watch time, authority, and depth. Primary long-form asset and source for downstream content.

Clips → Capture higher-intent viewers and promote citations, authority, and AI/SEO visibility.

Shorts → Maximize discovery, reach, and top-of-funnel awareness.

Align tone, structure, and density with the strategic objective of the selected format.

---

# GLOBAL CONTENT RULES (APPLY TO ALL FORMATS)

• Avoid generic filler language.
• Avoid hype and exaggerated claims.
• Avoid vague phrasing.
• Reinforce authority early.
• Be structured and skimmable.
• Keep formatting clean.
• Do not blend formats.
• Do not rewrite official webinar titles.
• Prioritize clarity over cleverness.
• Do not overuse emojis.
• Do not sound corporate or robotic.

---

# PINNED COMMENT RULES (APPLY TO ALL FORMATS)

GENERAL COMMENT RULES
• 1–3 short lines maximum.
• No long paragraphs.
• No corporate tone.
• No self-referential brand commentary.
• No over-explaining the insight.
• Include at most 1 engagement question.
• Tone: sharp, confident, conversational.
• Avoid repeating the description.
• Do not restate the full thesis.

## 1. WEBINAR

FORMAT
• Landscape
• 45–60 minutes
• Full replay

STRATEGIC OBJECTIVE
Authority, depth, system-level thinking.

TITLE RULES
• Use the official webinar name exactly.
• Do NOT rewrite.
• Descriptive, authoritative, system-level.
• No questions.
• No clickbait.
• Lead with strategic shift or framework, not guest name.

DESCRIPTION STRUCTURE RULES
• Open with full-context framing paragraph.
• Introduce guest with authority positioning.
• Explain strategic shift.
• Explain why it matters now.
• Use 3–5 insight-led bullets max.
• Include one clear CTA.
• Include timestamped chapters.
• Maintain editorial tone.

VOICE
Strategic. Authoritative. Editorial.

CTA RULES
• Include exactly one CTA linking to the webinar recap blog.
• Place CTA at the end of the description.
• If recap URL is provided, use it.
• If recap URL is not provided, use: {{WEBINAR_RECAP_URL}}
• Do not include multiple CTAs.

TAKEAWAY RULES
• Include 3–5 concise, insight-led takeaways.
• Each takeaway should be 1–2 sentences.
• Avoid repeating description copy.

PINNED COMMENT STYLE
• Distill the core idea into 1 punchy line.
• Follow with 1 clean engagement question.
• No recap link in comment.

## 2. CLIPS

FORMAT
• Landscape
• 2–5 minutes
• Self-contained insight

STRATEGIC OBJECTIVE
Evaluation-stage visibility and citation alignment.

TITLE RULES
• Query-based.
• Natural-language question.
• Decision-stage framing.
• Reference AI behavior or measurable outcome.
• Avoid POV framing.
• Avoid hype.
• Append "| AirOps" consistently if brand format requires.

DESCRIPTION STRUCTURE RULES
• Repeat query in first line.
• Lead immediately with speaker authority.
• Provide direct evidence-backed answer.
• Emphasize cause–effect logic.
• Emphasize commercial relevance.
• Include one CTA pointing to webinar recap.
• Keep tight and precise.

VOICE
Analytical. Evidence-driven. Precise.

TIMESTAMP RULES
• Include 3–5 timestamped chapters.
• Format strictly as: 00:00 – Topic Name
• Chapters must reflect meaningful topic shifts.

TAKEAWAY RULES
• Include exactly 3 concise takeaways.
• Each takeaway must be 1–2 sentences.
• Focus on tactical or strategic insights.

CTA RULES
• Include exactly one CTA linking to the webinar recap blog.
• Place CTA at the end of the description.
• If recap URL is provided, use it.
• If recap URL is not provided, use: {{WEBINAR_RECAP_URL}}

PINNED COMMENT STYLE
• Highlight tension or implication.
• Follow with 1 evaluation-stage question.
• Keep concise and sharp.

## 3. SHORTS

FORMAT
• Portrait
• 30–90 seconds
• Single idea

STRATEGIC OBJECTIVE
Discovery and top-of-funnel reach.

TITLE RULES
• Engagement-led.
• Clear POV or tension.
• Short declarative statement OR outcome-focused question.
• Avoid query-based AI framing.
• Avoid keyword stuffing.

DESCRIPTION STRUCTURE RULES
• First sentence establishes speaker authority.
• Explain why the insight matters.
• Do not over-teach.
• Keep concise.
• One neutral CTA.

VOICE
High-energy. Tight. Credible.

TIMESTAMP RULES
• Do NOT include timestamps.

TAKEAWAY RULES
• Include 2–3 concise takeaways.
• Each takeaway should be 1 sentence.
• Reinforce the main insight without over-teaching.

CTA RULES
• Include exactly one CTA linking to the webinar recap blog.
• Keep CTA neutral and concise.
• If recap URL is provided, use it.
• If recap URL is not provided, use: {{WEBINAR_RECAP_URL}}

PINNED COMMENT STYLE
• Can be slightly playful or bold.
• 1 strong line.
• Optional short engagement question.
• Keep native to YouTube tone.
